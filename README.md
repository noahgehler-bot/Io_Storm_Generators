As a part of the 2026 Gustavus RadioJOVE research team, these programs* are created to generate synthetic data of Jupiter-Io storms.

The noise backround is provided in AlgoNoise.py and is used across all storm types. All Io-A, B, and C storm generators are modeled after data collected and displayed in 
Richard S. Flagg's Listening to Jupiter: A Guide for the Amateur Radio Astronomer. Io-D was modeled after data displayed in the Morphology of the Jupiter Io-D Decametric Radio Source paper. We hope future research can utilise these generators and write a code that can effectively 
classify recorded data to its appropiate storm type. This will further our understanding of the Jupiter-Io magnetic interaction. From there, we hope that a similar program 
can be built and applied towards exomoon candidates. This program could distinguish noise from emissions and lead to the confirmation of an exomoon. We have taken the first 
step towards a tool that could aid in the search for an exomoon and we hope our vision for this tool can one day become a reality.
